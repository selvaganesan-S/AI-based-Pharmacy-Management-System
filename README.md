# AI-Based Pharmacy Management System

## Project Description
The AI-Based Pharmacy Management System is a Java desktop application developed using Java, JDBC, and MySQL. It helps manage pharmacy operations efficiently with AI-based features such as medicine recommendations, low stock alerts, expiry alerts, and sales analysis.

## Features
- Medicine Management (CRUD)
- Customer Management (CRUD)
- Supplier Management (CRUD)
- Sales Management
- Low Stock Alert
- Expiry Alert
- AI-Based Medicine Recommendation

## Technologies Used
- Java
- JDBC
- MySQL
- Eclipse IDE

## Future Enhancements
- ChatGPT API Integration
- Sales Prediction using AI
- Barcode Scanner Support
- Email/SMS Notifications
